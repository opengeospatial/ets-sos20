Overview

This test suite is based on the following OGC specifications:

  • OGC Web Services Common Specification, Version 1.1.0 [OGC 06-121r3]
  • Definition Identifier URNs in OGC Namespace, Version 1.1.0 [OGC 06-023r1] (ISO/CD 19136, OGC 03-105r1)
  • OpenGIS Sensor Observation Service Implementation Specification, Version 2.0 [OGC 12-006]

What is tested

  • GetCapabilities, GET / POST / SOAP method
  • DescribeSenor, GET / POST / SOAP method
  • GetObservation, GET / POST / SOAP method

What is not tested

  • Optional operations:
    • GetObservationByID
    • GetFeatureOfInterest
    • InsertSensor
    • DeleteSensor
    • InsertObservation
    • InsertResult
    • InsertResultTemplate
    • GetResultTemplate
    • GetResult

Test Data

  • n/a

Namespaces

  • Service being tested must use these namespaces:

    fes-http://www.opengis.net/fes/2.0
    gml-http://www.opengis.net/gml/3.2
    om-http://www.opengis.net/om/2.0
    ows-http://www.opengis.net/ows/1.1
    soap11-http://schemas.xmlsoap.org/soap/
    soap12-http://www.w3.org/2003/05/soap-envelope
    sos-http://www.opengis.net/sos/2.0
    swe-http://www.opengis.net/swe/2.0
    swes-http://www.opengis.net/swes/2.0
    wsa-http://www.w3.org/2005/08/addressing
    xs-http://www.w3.org/2001/XMLSchema
