Sensor Observation Service (SOS) 2.0 Conformance Test Suite

Scope

This test suite verifies conformance with Sensor Observation Service (SOS) 2.0. It is based on the following OGC specifications:

  • OpenGIS Sensor Observation Service Implementation Specification, Version 2.0 [OGC 12-006]
  • OGC Web Services Common Specification, Version 1.1.0 [OGC 06-121r3]
  • Definition Identifier URNs in OGC Namespace, Version 1.1.0 [OGC 06-023r1] (ISO/CD 19136, OGC 03-105r1)

Minimum Conformance classes to pass the test

To pass the test it is necessary to pass the following conformance classes:

  • Pass the SOS Core
  • Pass the SOS KVP Binding Extension or SOS SOAP Binding

What is tested

  • GetCapabilities, GET / POST / SOAP method
  • DescribeSenor, GET / POST / SOAP method
  • GetObservation, GET / POST / SOAP method
  • GetObservationByID, GET / POST / SOAP method
  • GetFeatureOfInterest, GET / POST / SOAP method
  • GetResultTemplate, GET / POST / SOAP method
  • GetResult, GET / POST / SOAP method

What is not tested

  • InsertSensor, GET / POST / SOAP method
  • DeleteSensor, GET / POST / SOAP method
  • InsertObservation, GET / POST / SOAP method
  • InsertResult, GET / POST / SOAP method
  • InsertResultTemplate, GET / POST / SOAP method
